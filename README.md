# 停车计时 ParkTimer

一个帮助您记录停车位置、追踪停车时长、提前提醒整点的Web应用。

## 功能特点

- 📷 **拍照记录** - 拍摄停车位照片，方便找车
- ⏱️ **实时计时** - 显示已停车时长
- 🔔 **整点提醒** - 提前5分钟通知，避免超时

## 使用方法

1. 点击"开始停车"
2. 拍摄停车位照片
3. 自动开始计时
4. 接近整点时收到提醒
5. 取车后点击"结束停车"

## 技术栈

- HTML5 + CSS3 + JavaScript
- PWA支持（离线可用）
- Web Notifications API
